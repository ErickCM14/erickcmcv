Proyecto Angular

Fecha de elaboración: Febrero 2021 - Actual

Proyecto en elaboración de estadia de Residencias Profesionales, se utiliza Angular como Frontend y Firebase como base de datos, manejando la Authentication de Firebase y solicitudes HTTP mediante el uso de Observables para el manejo y manipulación de datos en formato JSON con la API REST de Firebase, se implenta el envio de correo automatizado para aviso de servicios de las cuentas de clientes que estan por vencer dadas las fechas de vencimiento en la sección de "Cuentas" y también Backups al correo quincenales de la base de datos de Firebase, usando Nodemailer para envio de correo y Express para el manejo de rutas. Totalmente Responsive. Se uso POSTMAN para el desarrollo Git y Github para el manejo de versiones.

Tambien el proyecto integra extracción de datos de Google Analytics en la sección SEO, para esto hay que tener permisos a la cuenta de correo espeficica.

El proyecto esta en desarrollo con funcionalidades por mejorar frontend y backend.
Se subió a producción y se hicieron pruebas con el envio de correo, el código de envio de correo se subio a Heroku un servidor en la nube para comprobar que se hace correctamente, dado que en el Hosting contratado por la emrpesa no adquirio la posibilidad de correr apps en NodeJS.

NOTA: No se puede recargar la página porque manda un error 404, tampoco entrar a la sección SEO y regresar dado que es un documento HTML y provoca recargo de página, esto lo provoca el Hosting gratuito, esta probado en producción en un Hosting de paga por la empresa y funciona correctamente.
